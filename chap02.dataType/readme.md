# data structure

Rust 中每一个值都属于某一个 数据类型（data type），以便明确数据处理方式。
数据类型既可以这样划分为两个子集：[The Rust Programming Language](https://doc.rust-lang.org/book/ch03-02-data-types.html)

- 标量（scalar）
- 复合（compound）

也可以这样划分：[The Rust Reference](https://doc.rust-lang.org/reference/types.html)

Primitive types:

```
Boolean — bool
Numeric — integer and float
Textual — char and str
Never — ! — a type with no values
```

Sequence types:

```
Tuple
Array
Slice
```

User-defined types:

```
Struct
Enum
Union
```

Function types:

```
Functions
Closures
```

Pointer types:

```
References
Raw pointers
Function pointers
```

Trait types:

```
Trait objects
Impl trait
```
